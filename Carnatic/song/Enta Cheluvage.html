  Lyrics - English:

PallavieMthA celuvage magaLanu koTTanu girirAjanu nODammamma
AnupallavikaMbuhara Siva celuvanennuta meccidanu tA nODammamma
Caranam 1mOre jatu mUru kaNNa viparItava nODammamma
koraLoLu ruMDa mAleya dharisida uraga bhUshhaNana nODamma //
Caranam 2taleyeMbOdu nODidare jaDe hoLeyutide nODammamma
halavu kAlada tapasi rudrana mai bUdiyu nODammamma //
Caranam 3bhUta prEta piSAcigaLella parivAravu nODammamma
Itana nAmavu oMdE maMgaLa muppura harana nODammamma //
Caranam 4maneyeMbudu smaSAnavu nODe gaja carmAMbaravammamma
haNavoMdAdaru kaiyoLagilla kapparavanu nODammamma //
Caranam 5naMdi vAhana nIlakaMThana nirguNana nODamammamma
iMdira ramaNa SrI puraMdara viThalana poMdidavana nODammamma //
