( shankaraabharaNam )

( mOhanam )
Caranam 2vasantanum madanum vAridhiyum kuyilum matiyum sati sheyya vADugirAL murugA
( vasantaa )
Caranam 3nIyE sakhAnAga niccayamAga ninaivu koNDOngi nirkkirAL ArumukhA
( sahaanaa )
Caranam 4ennaik-kaNDAlunnai azhaittODi vAvenbAl pinnEyoruvarODum pEshavum nANuvAL
( tODi )
Caranam 5unakkishaidA-nAyaki ival tAnenru urudiyAgak-koLLuvIr umai maindA
( naayaki )
Caranam 6kannaDamum vaDa mozhiyum shetamizhum kaTrarinda kaNmaNiyAL innishaiyil kinnariyAL anna naDai minniDaiyAL un ninaivuDaiyAL
( kaanaDaa )
Caranam 7avaLin nEsham sArangAN avaniyil uvamai shollavoNNAda uttami siddhini
( saaranga )
Caranam 8shrIkari ivaLai nI shErndu vAzhvAi tiruccendil murugA harikEshan kumArA
( shree raagam )
